# 🚀 DevOps Job Automation System

## 📌 Overview
This project automates DevOps job discovery using Python and CI/CD pipelines.

## ⚙️ Tech Stack
- Python
- GitHub Actions
- Web Scraping (BeautifulSoup)

## 🔁 Workflow
1. Fetch jobs from Indeed
2. Filter DevOps roles
3. Store in CSV
4. Run daily via GitHub Actions

## 📈 Features
- Automated job tracking
- CI/CD pipeline
- Scheduled workflows
- Real-world DevOps use case

## 🎯 Outcome
Reduced manual job search effort by 80% using automation.
